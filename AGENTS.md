# AGENTS.md

## このプロジェクトについて

このプロジェクトは、自分用ホームページの制作途中のサイトです。
目的は、ポートフォリオ・自己紹介・制作物紹介に使える個人サイトを完成させることです。

## 作業方針

- 既存のデザインや構成を大きく壊さない
- 変更前に現在のファイル構成とコードを確認する
- いきなり全面リファクタリングしない
- 小さな単位で安全に修正する
- HTML / CSS / JavaScript の基本構成を優先する
- 見た目だけでなく、読みやすさ・保守性も重視する

## 優先順位

1. レイアウト崩れの修正
2. レスポンシブ対応
3. CSSの整理
4. HTML構造の改善
5. アニメーションや装飾の追加
6. SEOやOGPなどの調整

## デザイン方針

- シンプルで見やすいデザイン
- 余白をしっかり取る
- 色を使いすぎない
- スマホ表示を優先して確認する
- ボタンやリンクは押せることが分かる見た目にする

## コーディングルール

- HTMLは意味のあるタグを使う
  - header
  - main
  - section
  - footer
  - nav
- CSSはできるだけクラス名で管理する
- 同じCSSを何度も書かない
- 不要なコードやコメントアウトは削除する
- ファイル名・クラス名は分かりやすくする
- JavaScriptは必要最小限にする

## 禁止事項

- 既存ファイルを勝手に大量削除しない
- デザインを完全に別物にしない
- 使っていない外部ライブラリを勝手に追加しない
- 複雑すぎる実装にしない
- 本番公開用の個人情報を勝手に書かない

## 修正するときの進め方

1. 現在の問題点を確認する
2. 必要な変更内容を簡単に説明する
3. 最小限の修正を行う
4. 表示崩れがないか確認する
5. 変更点をまとめる

## 確認してほしい項目

- PC表示
- スマホ表示
- ヘッダー
- メインビジュアル
- 自己紹介セクション
- 制作物セクション
- お問い合わせ・リンク部分
- フッター

## 目指す完成形

IT学習者・Web制作者としての成長が伝わる、シンプルで実用的な個人サイト。
派手さよりも、見やすさ・信頼感・整理された構成を重視する。
